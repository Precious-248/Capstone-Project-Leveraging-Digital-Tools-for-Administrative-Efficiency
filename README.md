# Capstone-Project-Leveraging-Digital-Tools-for-Administrative-Efficiency

A comprehensive Capstone Project for the Pracvarsity Microdegree in Computer Essentials & Emerging Technologies. This project demonstrates how integrating AI-driven tools (with modern data platforms can transition workflows from manual, paper-based processes to efficient, data-driven systems.

## Tools & Technologies Utilized:

* **Research & Automation:** ChatGPT & Gemini (Strategic planning and workflow automation).
* **Visual Design:** Leonardo AI (Custom visuals and graphical assets).
* **Presentation:** Gamma (Slides creation and structured narrative).
* **Voice-Over:** ElevenLabs (AI-generated professional narration).
* **Data & Operations:** Microsoft Excel, Google Workspace, and Power BI.
